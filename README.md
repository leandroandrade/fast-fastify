# fast-fastify

Sample project using fastify.

Documentation: https://www.fastify.io/

Installation:

1. [nodejs.org](https://nodejs.org)
2. [npmjs.com](https://www.npmjs.com) (includes node)
3. git clone https://github.com/leandroandrade/fast-fastify
3. cd ./fast-fastify/
4. npm install
5. npm start
